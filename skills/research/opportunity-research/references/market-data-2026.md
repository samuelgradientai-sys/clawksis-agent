### Five product opportunities in code review (2026)
1. Framework-specific review (Rails, Django, Next.js)
2. Security-first PR review (buyer = AppSec team, integrate with SIEM)
3. Legacy/untested codebases (Java 8, COBOL, PHP 5)
4. Review for AI-generated PRs (understand Claude Code/Codex output)
5. Review observability (dashboards for engineering managers)

- Best pricing: fraction of what you replace (replace $3K/mo hire -> charge $299/mo)

## Claude Code Security Analysis (2026)
- Launched Feb 2026 as /security-review, then full Claude Code Security
- Found 500+ high-severity vulnerabilities in production OSS codebases
- Available only to Claude Enterprise/Team customers
- Key limitation: non-deterministic (same scan != same results)
- Only scans source code, does NOT test running applications
- False positive rates not published
- Prompt injection risk acknowledged by Anthropic
- Not hardened against adversarial inputs
- Source: StackHawk, Anthropic blog, Dark Reading, The Register
